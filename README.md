🛍️ Nikkita - Talento Tech Ecommerce

Proyecto de e-commerce responsive desarrollado con HTML5 y CSS3 inspirado en una tienda online de novelas de Agatha Christie.

La aplicación presenta un catálogo de libros clásicos con diseño adaptable a dispositivos móviles, navegación intuitiva y una interfaz moderna orientada a la experiencia del usuario.

✨ Características
* Diseño responsive 📱
* Navbar adaptable para mobile
* Hero section destacada
* Cards de productos
* Efectos hover y transiciones
* Footer completo con redes sociales
* Integración con Font Awesome
* Uso de Google Fonts
* Layout moderno utilizando Flexbox


🛠️ Tecnologías utilizadas
* HTML5
* CSS3
* Flexbox
* Media Queries
* Font Awesome
* Google Fonts

🚀 Funcionalidades principales

🧭 Navegación
* Menú desktop y mobile
* Íconos interactivos
* Header sticky

📚 Catálogo de productos
* Cards con imágenes
* Nombre y precio de productos
* Botón de agregar al carrito

🎨 Diseño responsive
* Adaptación automática para tablets y celulares
* Media queries para diferentes resoluciones
